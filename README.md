# Política de Privacidad - Gisionarios

Esta es una landing page básica con la política de privacidad de la empresa Gisionarios.

## Archivos

- `index.html`: Página principal (landing page) con enlace a la política de privacidad.
- `privacidad.html`: Página dedicada a la política de privacidad.

## Cómo usar

Abre `index.html` en tu navegador para ver la página principal.

## Subir a GitHub

1. Crea un nuevo repositorio en GitHub.
2. Sube los archivos `index.html`, `privacidad.html` y `README.md`.
3. Habilita GitHub Pages en la configuración del repositorio para publicar la página.
